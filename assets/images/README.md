# Logo Image

Please save the astronaut logo image as:
- **File name**: `astronaut-logo.png`
- **Location**: `/var/www/prompt-db.dainedvorak.com/assets/images/astronaut-logo.png`

The image should be the circular astronaut logo with the space background that was provided.

## Quick Setup

If you have the image file locally, you can copy it here with:
```bash
cp /path/to/your/astronaut-logo.png /var/www/prompt-db.dainedvorak.com/assets/images/astronaut-logo.png
```

Alternatively, you can use any image editor to save the astronaut logo image directly to this directory.
